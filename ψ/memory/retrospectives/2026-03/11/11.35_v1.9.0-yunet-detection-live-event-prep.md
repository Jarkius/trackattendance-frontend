The auto-venv addition to zshrc was a nice quality-of-life improvement. The user was hitting `ModuleNotFoundError: No module named 'PyQt6'` because they were using system Python. The chpwd hook pattern is clean and project-aware.

## Lessons Learned
- **Pattern**: Qt overlay positioning needs `frameGeometry()` for absolute screen position, `geometry()` for content-relative — always compute title bar delta when placing floating widgets
- **Discovery**: Both export paths (attendance + dashboard) should use the same timezone formatting — always cross-check all code paths that output timestamps
- **Pattern**: Auto-venv via `chpwd` hook is the cleanest approach for multi-project Python workflows
